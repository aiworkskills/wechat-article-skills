# 首次引导

当项目和用户目录下均不存在 `config.yaml` 时，触发首次引导。

## 流程

### 第1步：确定保存位置

询问用户：
- **项目级**（`.aws-article/config.yaml`）：仅当前项目生效
- **用户级**（`~/.aws-article/config.yaml`）：所有项目通用

### 第2步：收集配置

分 3 组询问，每组可合并为一次提问：

**第一组：账号定位**
- 账号类型/领域（如科技、职场、情感）
- 目标读者
- 语气调性
- 更新频率
- 默认作者名

**第二组：内容偏好**
- 选题方向或禁区
- 标题风格偏好
- 文章风格
- 文末固定引导语（如关注提示、往期推荐）

**第三组：排版与图片**
- 默认排版预设（可跳过，后续选择）
- 封面比例与风格
- 配图密度

未回答的项使用默认值。

### 第3步：写入配置

将用户选择写入 `config.yaml`，格式参照 [config-schema.md](config-schema.md)。

### 第4步：确认并继续

告知用户「配置已保存到 [路径]，后续可直接编辑该文件修改」，然后继续执行用户原本的请求。
