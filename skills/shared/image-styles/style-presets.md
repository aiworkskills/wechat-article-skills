# 风格预设

按文章类型一键选择 Type + Style 组合。用户也可手动覆盖其中任一维度。

## 按文章类型

### 科技与技术

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `科技解析` | 信息图 | 蓝图 | API 文档、系统指标、技术深度 |
| `系统设计` | 框架 | 蓝图 | 架构图、系统设计 |
| `工具评测` | 对比 | 扁平矢量 | 工具对比、框架评测 |

### 知识与教程

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `知识科普` | 信息图 | 扁平矢量 | 概念解释、教程、入门指南 |
| `操作指南` | 流程图 | 简约线条 | 分步教程、操作流程 |
| `效率工具` | 信息图 | 简约线条 | 产品指南、SaaS 介绍 |

### 数据与分析

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `数据报告` | 信息图 | 商务 | 数据新闻、指标报告 |
| `趋势对比` | 对比 | 扁平矢量 | 行业对比、趋势分析 |

### 故事与情感

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `故事叙述` | 氛围 | 温暖 | 个人随笔、成长故事 |
| `生活方式` | 氛围 | 水彩 | 旅行、生活、文艺 |

### 观点与评论

| 预设 | Type | Style | 适用 |
|------|------|-------|------|
| `观点文章` | 氛围 | 海报 | 社论、评论、犀利观点 |

## 文章内容 → 预设推荐

根据文章类型（由 topics 或 writing 确定）自动推荐：

| 文章内容信号 | 推荐预设 | 备选 |
|-------------|---------|------|
| API、代码、技术、开发 | `科技解析` | `系统设计` |
| 教程、入门、指南 | `知识科普` | `操作指南` |
| 对比、评测、选择 | `工具评测` | `趋势对比` |
| 数据、报告、指标 | `数据报告` | `趋势对比` |
| 故事、经历、个人 | `故事叙述` | `生活方式` |
| 观点、评论、争议 | `观点文章` | — |
| 工具、产品、SaaS | `效率工具` | `知识科普` |
| 流程、步骤、操作 | `操作指南` | `知识科普` |
