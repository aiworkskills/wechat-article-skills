# 提交到公众号后台

## 手动方式

1. 登录微信公众平台（mp.weixin.qq.com）。  
2. 进入「素材管理」或「草稿箱」→ 新建图文消息。  
3. 将已排版的正文（含 HTML 或富文本）粘贴到正文区；标题、摘要、封面、作者等按后台表单填写。  
4. 正文内图片在后台逐一上传并插入；或使用「导入 Word/HTML」等能力（以平台当前功能为准）。  
5. 保存为草稿或直接群发；需开评论/仅粉丝可评时在设置中勾选。
