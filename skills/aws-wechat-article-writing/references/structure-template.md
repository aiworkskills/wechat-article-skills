# 公众号长文结构模板

## 标准结构

```
标题
├── 摘要（80-150字，概括核心价值）
├── 开头（2-3段，吸睛切入）
├── 正文主体
│   ├── 小标题一 + 段落
│   ├── 小标题二 + 段落
│   ├── 小标题三 + 段落
│   └── …（按内容需要增减）
├── 结尾（1-2段，总结 + 金句或行动号召）
└── 文末区块（引导关注、往期推荐等）
```

## 开头写法

| 类型 | 适用场景 | 示例 |
|------|---------|------|
| 提问式 | 痛点型文章 | 你有没有遇到过这样的情况：XXX？ |
| 场景式 | 故事型文章 | 上周和朋友聊天，他问了我一个问题…… |
| 数据式 | 干货/趋势型 | 据统计，80% 的人在 XXX 时会犯同一个错误。 |
| 金句式 | 观点型文章 | 有人说「XXX」，但我觉得事实恰恰相反。 |
| 直给式 | 教程/清单型 | 今天分享 5 个立刻能用的 XXX 技巧。 |

## 正文节奏

- **短段为主**：每段 3-5 行，手机阅读友好
- **节奏变化**：长短段交替，避免全篇节奏单一
- **每节一图**：配合 aws-wechat-article-images 预留配图位置
- **过渡句**：小标题之间用过渡句衔接，避免割裂感

## 结尾写法

| 类型 | 示例 |
|------|------|
| 总结型 | 总结一下，XXX 的关键就是这 3 点：…… |
| 金句型 | 与其等待完美时机，不如现在就开始行动。 |
| 行动号召 | 如果你也想试试，从今天开始做这一件事就够了。 |
| 互动型 | 你在 XXX 方面有什么经验？欢迎留言分享。 |

## 文末区块

按配置 `closing_block` 填充，常见内容：
- 关注引导：「觉得有用？点个关注，下次不迷路」
- 往期推荐：链接 2-3 篇相关文章
- 作者介绍：一句话介绍
- 转载声明：原创/转载标注
