# 审稿输出模板

## 分块详细格式

```markdown
## 审稿结果

### 标题
- [x] 字数：22 字 ≤ 25 字上限 ✅
- [ ] 发现禁用套路「建议收藏」 ❌
- 修改建议：去掉「建议收藏」，改为更具体的价值承诺

### 摘要
- [x] 字数：95 字，在 80-150 范围内 ✅
- [x] 与正文一致 ✅

### 正文
- [x] 敏感词检查通过 ✅
- [ ] 第 3 段发现错别字「帐号」→「账号」 ❌
- [x] 原创标注已添加 ✅
- 修改建议：修正第 3 段错别字

### 图片
- [x] 封面图已指定 ✅
- [x] 正文配图位置合理 ✅

### 链接
- [x] 无外链 ✅

## 总结
- 🔴 必须修改：标题去掉禁用套路、正文修正错别字
- 🟡 建议修改：无
- 修改后可进入排版与配图
```

## 简要清单格式

```markdown
## 审稿结果（简要）

| 检查项 | 结果 | 说明 |
|--------|------|------|
| 标题长度 | ✅ | 22/25 字 |
| 标题套路 | ❌ | 含「建议收藏」 |
| 摘要 | ✅ | 95 字 |
| 敏感词 | ✅ | 通过 |
| 错别字 | ❌ | 「帐号」→「账号」 |
| 原创标注 | ✅ | 已添加 |
| 图片 | ✅ | 封面+配图齐全 |
| 链接 | ✅ | 无外链 |

**必须修改**：标题去掉禁用套路、正文修正错别字
```
