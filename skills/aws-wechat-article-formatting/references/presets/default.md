# 简约科技

> 清爽简约的科技风排版，适合干货、教程、互联网话题类文章。

## 基础设置

| 属性 | 值 |
|------|-----|
| 正文字号 | 16px |
| 正文颜色 | #333333 |
| 正文行高 | 1.8 |
| 字间距 | 0.5px |
| 段间距 | 1.5em |
| 页面边距 | 左右 16px |
| 强调色 | #0F4C81（经典蓝） |
| 链接颜色 | #576B95（微信蓝） |

## 标题样式

| 级别 | 样式 |
|------|------|
| H1（文章标题） | 22px 加粗，居中，底部 1px 分割线 |
| H2（小标题） | 18px 加粗，强调色左边框 3px，左缩进 8px |
| H3（子标题） | 16px 加粗，强调色文字 |

## 正文样式

| 元素 | 样式 |
|------|------|
| 段落 | 首行不缩进，段间 1.5em |
| 加粗 | 强调色加粗 |
| 引用块 | 左边框 3px 强调色，背景 #F7F7F7，内边距 12px |
| 有序列表 | 强调色序号 |
| 无序列表 | 强调色圆点 |

## 特殊元素

| 元素 | 样式 |
|------|------|
| 分割线 | 居中虚线，宽 60%，颜色 #CCCCCC |
| 图注 | 14px，#999999，居中 |
| 文末区块 | 上方分割线，14px，#666666 |
