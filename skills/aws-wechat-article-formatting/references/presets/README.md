# 排版预设格式说明

本目录下每个 `.md` 文件为一套排版预设，文件名即预设名（如 `default.md` 对应预设名 `default`）。

## 预设文件结构

每个预设文件应包含以下部分：

```markdown
# [预设名称]

> 一句话描述风格与适用场景

## 基础设置
字号、字体、行高、颜色等基础参数。

## 标题样式
各级标题的样式规则。

## 正文样式
段落、列表、引用、代码等样式规则。

## 特殊元素
分割线、图注、文末区块等样式规则。
```

## 添加新预设

1. 在本目录新建 `<预设名>.md` 文件
2. 按上述结构填写排版规则
3. 在 SKILL.md 的预设表格中添加条目（可选）
