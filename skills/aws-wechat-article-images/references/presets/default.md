# 简约扁平

> 简约扁平风格，适合科技、互联网、通用类文章配图。

## 风格描述

扁平化设计，几何图形与线条组合，色彩明快但不过度，整体干净利落。适合干货型、教程型、科技资讯类文章。

## 封面 Prompt 模板

```
创建一张微信公众号封面图。
主题：{article_title}
风格：简约扁平插画，几何图形与线条
比例：{cover_aspect}
色调：以 {primary_color} 为主色，搭配白色和浅灰
要求：
- 画面简洁，留白充足
- 可包含与主题相关的抽象图形元素
- 不要包含文字（标题会单独叠加）
- 适合作为公众号封面
```

## 正文配图 Prompt 模板

```
创建一张微信公众号正文配图。
内容：{section_description}
风格：与封面统一的简约扁平风格
宽度：1080px
要求：
- 与正文段落内容呼应
- 简洁不喧宾夺主
- 色调与封面保持一致
```

## 默认参数

| 参数 | 默认值 |
|------|--------|
| 主色 | #0F4C81（经典蓝） |
| 辅色 | #F7F7F7（浅灰） |
| 构图 | 居中对称 |
| 元素 | 几何图形、线条、图标 |
