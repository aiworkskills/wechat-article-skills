# 配图风格预设格式说明

本目录下每个 `.md` 文件为一套配图风格预设，文件名即预设名（如 `default.md` 对应预设名 `default`）。

## 预设文件结构

每个预设文件应包含以下部分：

```markdown
# [预设名称]

> 一句话描述风格与适用场景

## 风格描述
视觉风格的文字描述（用于 prompt 生成）。

## 封面 Prompt 模板
用于生成封面图的 prompt 模板，含占位符。

## 正文配图 Prompt 模板
用于生成正文配图的 prompt 模板。

## 参数
颜色、构图、元素等默认参数。
```

## 添加新预设

1. 在本目录新建 `<预设名>.md` 文件
2. 按上述结构填写风格描述和 prompt 模板
3. 在 SKILL.md 的预设表格中添加条目（可选）
