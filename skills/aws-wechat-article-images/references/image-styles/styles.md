# 风格库

## Type（画面构成）

12 种图片类型，描述"画什么"：

| Type | 名称 | 说明 | 适用场景 |
|------|------|------|---------|
| `concept` | 概念讲解 | 视觉隐喻把抽象概念具象化 | 原理解释、概念科普、隐喻 |
| `process` | 流程步骤 | 分步引导图，1-2-3 清晰明了 | 教程、操作指南、工作流 |
| `whiteboard` | 板书白板 | 像老师在白板上手写画图 | 知识科普、原理讲解、课程笔记 |
| `data-viz` | 数据可视化 | 数字变图表，让数据说话 | 趋势分析、统计、报告 |
| `comparison` | 对比说明 | 左右/上下并排，一眼看清差异 | 方案对比、产品PK、改版前后 |
| `architecture` | 架构图 | 分层结构图，展示组成关系 | 系统架构、产品架构、技术栈 |
| `mindmap` | 思维导图 | 中心主题向外发散分支 | 知识框架、主题概览、关系梳理 |
| `timeline` | 时间线 | 按时间轴排列事件或里程碑 | 发展历程、路线图、年度总结 |
| `checklist` | 清单图 | 条目化要点配图标 | Tips、注意事项、最佳实践 |
| `quote-card` | 金句卡片 | 突出核心观点或名言金句 | 金句、核心结论、截图转发 |
| `scene` | 场景还原 | 还原真实使用场景，代入感 | 用户场景、产品展示、生活方式 |
| `atmosphere` | 氛围烘托 | 营造情绪氛围，视觉呼吸 | 情感渲染、散文配图、阅读节奏 |

封面图不属于以上 Type，单独处理（见 prompt-construction.md）。

## Style（视觉风格）

14 种视觉风格，描述"怎么画"。

### 核心风格（快速选择用）

| 核心风格 | 对应详细风格 | 适用场景 |
|---------|------------|---------|
| `扁平矢量` | flat-vector | 知识科普、教程、科技 |
| `简约线条` | notion-line | 通用、知识分享、SaaS |
| `蓝图` | blueprint | AI、前沿科技、系统设计 |
| `手绘` | hand-drawn | 轻松、个人成长、日常 |
| `水彩` | watercolor | 生活、情感、旅行、文艺 |
| `海报` | screen-print | 观点、评论、文化、社论 |

多数场景用核心风格即可。

### 完整风格库

| 风格 | 描述 | 适用场景 |
|------|------|---------|
| `flat-vector` | 扁平矢量，清晰几何形状 | 知识科普、教程、科技 |
| `notion-line` | 极简手绘线条 | 知识分享、效率工具 |
| `blueprint` | 技术蓝图风 | 架构设计、系统设计 |
| `hand-drawn` | 手绘涂鸦风 | 轻松、随笔 |
| `watercolor` | 水彩柔和 | 生活方式、情感 |
| `screen-print` | 海报丝印风 | 观点、社论 |
| `warm` | 温暖亲切 | 个人成长、教育 |
| `minimal` | 极简留白 | 哲学、极简主义 |
| `elegant` | 精致商务 | 商业、思想领导力 |
| `editorial` | 杂志信息图 | 数据新闻、深度报道 |
| `scientific` | 学术精确图 | 学术、研究 |
| `sketch` | 铅笔素描风 | 头脑风暴、创意 |
| `pixel-art` | 像素复古 | 游戏、复古科技 |
| `playful` | 活泼卡通 | 趣味、轻松科普 |
